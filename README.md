# Smart Campus Operations (MERN)

AI-powered Smart Scheduling and Campus Operations System.

## Stack
- MongoDB Atlas
- Express.js + Node.js
- React (Vite)
- Tailwind CSS
- Axios
- React Router
- JWT Authentication

## Run Locally

### 1) Backend
```bash
cd backend
copy .env.example .env
# update MONGO_URI and JWT_SECRET in .env
npm install
npm run dev
```

### 2) Frontend
```bash
cd frontend
copy .env.example .env
npm install
npm run dev
```

## Key Features
- Automated timetable generation
- Classroom allocation optimization
- Faculty workload balancing
- Laboratory scheduling
- Campus transportation optimization
- Predictive maintenance monitoring

## API Base URL
`http://localhost:5000/api`
